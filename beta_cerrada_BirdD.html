Bird Dash INICIA SU BETA CERRADA, DESTALLES Y REQUISITOS PARA PARTICIPAR

HBStudios presenta la beta cerrada para Bird Dash, conoce cómo puedes participar y ser parte de esta prueba cerrada:

Requisitos:

- SO Andorid 6 o superior.
- RAM: 2 GB o mas
- Espacio disponible: 20 MB
- Procesador: Gama baja o superior (Esto no afecta el rendimiento).

Codiciones para participar:

Debes contar con una cuenta de correo gmail para ser dado de alta, debes ser mayor de 16 años, debes saber que el uso del correo es meramente para habilitar el permiso y darte acceso al enlace de descarga no se usa para nada mas, no requerimos información personal ni datos de pago, es completamente gratis.
